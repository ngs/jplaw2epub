# jplaw2epub

Command line tool to convert [Japanese Standard Law XML Schema (法令標準XMLスキーマ)][xmldoc] into ePub file.

```sh
go install go.ngs.io/jplaw2epub@latest

jplaw2epub -d mylaw.epub path/to/law.xml
```
